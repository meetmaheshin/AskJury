# Favicon Generation Instructions

## Current Setup

A basic SVG favicon has been created at `favicon.svg`.

## To Create Professional Favicons:

### Option 1: Use Online Tool (Recommended)

1. **Visit**: https://realfavicongenerator.net/
2. **Upload** your logo or use the current `favicon.svg`
3. **Download** the generated package
4. **Extract** and copy all files to this `public/` folder
5. Files you'll get:
   - favicon.ico
   - favicon-16x16.png
   - favicon-32x32.png
   - apple-touch-icon.png
   - android-chrome-192x192.png
   - android-chrome-512x512.png

### Option 2: Create Custom Logo

If you want a custom design:

1. **Design** your logo (use Figma, Canva, or Photoshop)
2. **Export** as SVG or high-res PNG (512x512px minimum)
3. **Use** RealFaviconGenerator.net to convert to all sizes
4. **Copy** generated files here

### Option 3: Use the SVG Directly

The current `favicon.svg` will work for modern browsers.
For broader compatibility, generate PNG versions using Option 1.
